Detailed Description

Markup extension that parses Markdown/Obsidian-style images.

Supported syntax:

<img src="image.png" alt="Alt text"/>
<img src="image.png" alt="Alt text"/>

The alt text is extracted from the brackets [], the image source from parentheses (), and optional attributes can be provided in curly braces {}.
