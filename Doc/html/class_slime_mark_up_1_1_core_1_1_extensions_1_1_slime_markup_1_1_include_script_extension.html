Detailed Description

Markup extension that allows including external JavaScript files using a special comment syntax.

Supported syntax:

The path is resolved to an absolute path. If the file exists, a script element is generated with a src attribute. Otherwise, an HTML comment indicating an error is returned.

This extension has priority and is processed before other extensions.
