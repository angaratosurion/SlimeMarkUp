Detailed Description

Represents an element in a markup structure, including its tag name, content, attributes, and child elements.

Use this class to model hierarchical markup such as HTML or XML. Each instance can contain nested child elements, allowing for complex document structures. The class does not perform validation or rendering; it is intended for representing markup data in memory.
