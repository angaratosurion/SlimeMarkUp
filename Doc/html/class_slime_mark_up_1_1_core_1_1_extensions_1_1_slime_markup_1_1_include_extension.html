Detailed Description

Markup extension that allows including external files into the document using a special comment syntax.

Supported syntax:

The path is resolved to an absolute path. If the file exists, its content is read and returned as a raw block inside MarkupElement.Content. Otherwise, an HTML comment indicating an error is generated.

This extension has priority and is processed before other extensions.
