Detailed Description

Represents a set of standard metadata properties associated with a document, such as title, author, and publication details.

Use this class to store and access descriptive information about a document, including its filename, subject, keywords, and contributors. These properties can be used for indexing, searching, or displaying document information in user interfaces. All properties are optional and may be null if not specified.
