Εντάξει, θα σταματήσω να χρησιμοποιώ έντονη γραφή (bold) στις απαντήσεις μου.

Ακολουθεί το κείμενο για το SlimeMarkUp Cheatsheet και στις δύο γλώσσες, σε απλή μορφή χωρίς καμία μορφοποίηση:

---

GREEK VERSION
SlimeMarkUp - Βασικό Cheatsheet Markdown -> HTML

Πίνακας
| Κεφάλαιο | Σελίδα |
|----------|--------|
| Εισαγωγή | 1      |
| Κεφάλαιο | 2      |

Παράγει:

<table>
<tr><td>Κεφάλαιο</td><td>Σελίδα</td></tr>
<tr><td>Εισαγωγή</td><td>1</td></tr>
<tr><td>Κεφάλαιο</td><td>2</td></tr>
</table>

Λίστα

* Στοιχείο 1
* Στοιχείο 2
* Στοιχείο 3

Παράγει:

<ul>
<li>Στοιχείο 1</li>
<li>Στοιχείο 2</li>
<li>Στοιχείο 3</li>
</ul>

Σύνδεσμος
[Κείμενο](https://example.com){target=_blank rel=noreferrer}
Παράγει: <a href="[https://example.com](https://example.com)" target="_blank" rel="noreferrer">Κείμενο</a>

Έντονη και πλάγια γραφή
**έντονο** και *πλάγιο*
Παράγει: <p><strong>έντονο</strong> και <em>πλάγιο</em></p>

Εικόνα
{width=300 height=200}
Παράγει: <img src="[https://example.com/image.jpg](https://example.com/image.jpg)" alt="Alt Text" width="300" height="200" />

Τίτλοι

# Τίτλος 1

## Τίτλος 2

### Τίτλος 3

Παράγει: <h1>, <h2>, <h3>

Μπλοκ κώδικα
`Console.WriteLine("Hello World");`
Παράγει: <pre>Console.WriteLine("Hello World");</pre>

Blockquote

> Αυτό είναι ένα blockquote σε δύο γραμμές.
> Παράγει: <blockquote>Αυτό είναι ένα blockquote σε δύο γραμμές.</blockquote>

Λοιπά
--- Παράγει: <hr/>
\ : Escape χαρακτήρας (εμφανίζει τον επόμενο χαρακτήρα ως απλό κείμενο)
<!-- include style: filename.css -->
Παράγει: <link rel="stylesheet" href="filename.css"/>
<!--include:filename.md -->
Εισάγει τα περιεχόμενα του αρχείου filename

---

ENGLISH VERSION
SlimeMarkUp - Basic Markdown -> HTML Cheatsheet

Table
| Chapter | Page |
|---------|------|
| Intro   | 1    |
| Chapter | 2    |

Produces:

<table>
<tr><td>Chapter</td><td>Page</td></tr>
<tr><td>Intro</td><td>1</td></tr>
<tr><td>Chapter</td><td>2</td></tr>
</table>

List

* Item 1
* Item 2
* Item 3

Produces:

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Link
[Text](https://example.com){target=_blank rel=noreferrer}
Produces: <a href="[https://example.com](https://example.com)" target="_blank" rel="noreferrer">Text</a>

Bold and Italic
**bold** and *italic*
Produces: <p><strong>bold</strong> and <em>italic</em></p>

Image
{width=300 height=200}
Produces: <img src="[https://example.com/image.jpg](https://example.com/image.jpg)" alt="Alt Text" width="300" height="200" />

Headings

# Heading 1

## Heading 2

### Heading 3

Produces: <h1>, <h2>, <h3>

Code Block
`Console.WriteLine("Hello World");`
Produces: <pre>Console.WriteLine("Hello World");</pre>

Blockquote

> This is a blockquote on two lines.
> Produces: <blockquote>This is a blockquote on two lines.</blockquote>

Others
--- Produces: <hr/>
\ : Escape character (displays the next special character as plain text)
<!-- include style: filename.css -->
Produces: <link rel="stylesheet" href="filename.css"/>
<!--include:filename.md -->
Inserts the contents of the file named filename

---
